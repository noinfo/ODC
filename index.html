ODC by noinfo

On Demand CMS

It is meant to be a quick, simple and highly adaptable solution to add editable regions to existing webpages.

Dependencies

Non-Optional: PHP, MySQL Optional: nicedit (www.nicedit.com) Optional: php-markdown

Install

Disclaimer: This software is provided as-is and I will not be held responsible for any damages that come from using this software. This software is written and meant to be in the back-end of a webpage, ideally behind a .htaccess to restrict access only to authorized parties. As such no sanitation of strings has been made when connecting to the database as it is assumed that only authorized persons can access the back-end (and that those won't try SQL-injection attacks, as they could simply delete all content using the edit functions).
